# AM Collision & Towing

Professional auto body restoration and emergency recovery website for AM Collision & Towing, serving the Bohemia, New York area since 2009.

## Tech Stack

- **React** + **TypeScript** — frontend UI
- **Vite** — build tool
- **Tailwind CSS** — styling
- **Supabase** — authentication and database (leads, profiles)
- **Framer Motion** — animations

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```

2. Create a `.env.local` file in the project root with your Supabase credentials:
   ```
   VITE_SUPABASE_URL=your_supabase_project_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

3. Run the development server:
   ```
   npm run dev
   ```
